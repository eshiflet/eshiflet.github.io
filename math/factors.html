Each card below shows an integer and all of the factor pairs that multiply to make it. An integer with only one factor pair, 1 and itself, is a prime number — those cards are highlighted in blue.

Explore primes visually

Want a more intuitive feel for why a number is prime? Check out Prime Waves, an interactive tool that tests integers for primality by overlapping cosine waves — if all the waves but one cancel out, the number is prime.
